# 📱 Overview

SmartBike Guardian is a modern Android application designed for shared mobility companies such as Yulu, Lime and Bird.

The application helps riders and fleet operators monitor bike health, report issues, validate parking locations and manage vehicle maintenance using an intuitive mobile interface.

# 🎯 Problem Statement

Shared mobility platforms face several operational challenges:

- Damaged bikes remain unnoticed
- Improper parking
- Low battery vehicles
- Manual maintenance tracking
- Poor fleet visibility

SmartBike Guardian provides a single platform to simplify these operations.
